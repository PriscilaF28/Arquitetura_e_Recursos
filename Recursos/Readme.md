## Configurando Recursos

Você tem a opção de criar uma VM com configurações já pre-definidas de acordo com a necesidade e com os recursos já criados na VM. São indicadas de forma detalhas as configurações e os custos referentes.

Para a criação manual, uma única maquina virtual apresenta 8 categorias de configurações, que devem ser consideradas.
* Criar o nome, identificar a região, 
* indicar o tipo de disponibilidade, ou seja a presença de redundância dos dados em outros servidores. 
* selecionar o tamanho de máquina, disco e memória. Os tamanhos das VMs são categorizados por tipo de finalidade de uso.

## Conjunto de Dimensionamento 
=> indica como pode ser feito o crescimento do recurso
* você pode indicar a quantidade inicial de instâncias, e configurar a quantidade mínima e máxima que pode usar
* determinar como deve ser feito o escalonamento horizontal de forma automática, considerando o mínimo e máximo editor
* definir as condições para aumento e redução de instâncias
* definir o tempo de verificação de cada atualização
